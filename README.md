# 🎬 Netflix Data Analysis Project
# 📌 Introduction

This project focuses on analyzing Netflix content data to uncover patterns, trends, and insights about movies and TV shows available on the platform. The analysis helps in understanding content distribution, popularity, and user preferences.

# 🎯 Objective
Analyze Netflix dataset using Python
Perform data cleaning and preprocessing
Explore trends in movies and TV shows
Visualize insights using graphs and charts
Build a foundation for dashboard creation (Power BI/Tableau)

# 📂 Dataset
Source: Netflix dataset (CSV format)
Contains information such as:
Title
Genre
Release Year
Rating
Duration
Country
Type (Movie/TV Show)

# 🛠️ Tools & Technologies
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook / VS Code

# 🔍 Steps Performed
Data Loading
Data Cleaning (handling null values, duplicates)
Data Transformation
Exploratory Data Analysis (EDA)
Data Visualization

# 📈 Visualizations
Bar charts for content type distribution
Count plots for ratings
Trend analysis by release year
Genre-wise analysis
